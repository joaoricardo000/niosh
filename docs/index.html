    Sobre esta tradução

    • Tradução para português do Brasil do NIOSH Pocket Guide to Chemical Hazards (2005-149, 3ª impressão).
    • Medidas convertidas para o sistema métrico (°F→°C, mmHg→kPa, atm→kPa), com o valor original entre parênteses.
    • Nomes químicos traduzidos para PT-BR (original em inglês entre parênteses). Fórmulas, CAS#, RTECS#, DOT# e códigos de respirador/medição mantidos no original.
    • Os índices finais (CAS/DOT/sinônimos) foram mantidos no original para preservar a função de busca alfabética.

    Documento original de domínio público do NIOSH. Esta é uma tradução automática; para uso crítico (resposta a emergência, decisões de segurança), confirme as informações na fonte oficial em cdc.gov/niosh/npg. Gerado em 2026-06-29.
