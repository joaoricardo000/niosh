## Tabela 2 — Códigos de Proteção Individual e Higiene

| Código | Definição |
|--------|-----------|
| **Pele:** | |
| Evitar contato com a pele | Vista roupas de proteção individual apropriadas para evitar o contato com a pele. |
| Congelamento (queimadura por frio) | Gases comprimidos podem gerar baixas temperaturas ao se expandirem rapidamente. Vazamentos e usos que permitam expansão rápida podem causar risco de congelamento (queimadura por frio). Vista roupas de proteção individual apropriadas para evitar o congelamento da pele. |
| N.R. | Não há recomendação especificando a necessidade de equipamento de proteção individual para o corpo. |
| **Olhos:** | |
| Evitar contato com os olhos | Use proteção ocular apropriada para evitar o contato com os olhos. |
| Congelamento (queimadura por frio) | Use proteção ocular apropriada para evitar o contato dos olhos com o líquido, que poderia resultar em queimaduras ou dano tecidual por congelamento. |
| N.R. | Não há recomendação especificando a necessidade de proteção ocular. |
| **Lavar a pele:** | |
| Quando contaminada | O trabalhador deve lavar a pele imediatamente quando esta for contaminada. |
| Diariamente | O trabalhador deve lavar-se diariamente ao final de cada turno de trabalho e antes de comer, beber, fumar, etc. |
| N.R. | Não há recomendação especificando a necessidade de remover a substância da pele (seja imediatamente ou ao final do turno de trabalho). |
| **Remover (roupa):** | |
| Quando molhada ou contaminada | Roupas de trabalho que ficarem molhadas ou significativamente contaminadas devem ser removidas e substituídas. |
| Quando molhada (inflam.) | Roupas de trabalho que ficarem molhadas devem ser removidas imediatamente devido ao seu risco de inflamabilidade (ou seja, para líquidos com ponto de fulgor <38°C (100°F)). |
| N.R. | Não há recomendação especificando a necessidade de remover roupas que fiquem molhadas ou contaminadas. |

## Tabela 2 (Continuação) — Códigos de Proteção Individual e Higiene

| Código | Definição |
|--------|-----------|
| **Trocar (roupa):** | |
| Diariamente | Os trabalhadores cujas roupas possam ter sido contaminadas devem trocá-las por roupas não contaminadas antes de deixar as instalações de trabalho. |
| N.R. | Não há recomendação especificando a necessidade de o trabalhador trocar de roupa após o turno. |
| **Disponibilizar:** | |
| Lava-olhos | Fontes lava-olhos devem ser disponibilizadas em áreas onde haja qualquer possibilidade de os trabalhadores serem expostos às substâncias; isso independe da recomendação relativa ao uso de proteção ocular. |
| Chuveiro de emergência | Instalações para ensopar rapidamente o corpo devem ser disponibilizadas dentro da área de trabalho imediata para uso emergencial onde houver possibilidade de exposição. [Nota: Pretende-se que essas instalações forneçam quantidade ou fluxo de água suficiente para remover rapidamente a substância de quaisquer áreas do corpo que possam ser expostas. A determinação efetiva do que constitui uma instalação de chuveiro de emergência adequada depende das circunstâncias específicas. Em certos casos, um chuveiro de dilúvio deve estar prontamente disponível, ao passo que em outros a disponibilidade de água de uma pia ou mangueira poderia ser considerada adequada.] |
| Lavagem para congelamento (frostbite wash) | Chuveiros de emergência e/ou fontes lava-olhos devem ser disponibilizados dentro da área de trabalho imediata para uso emergencial onde houver qualquer possibilidade de exposição a líquidos extremamente frios ou de rápida evaporação. |
| **Outros códigos:** | |
| Liq | Líquido |
| Molt | Fundido (molten) |
| Sol | Sólido |
| Soln | Solução contendo o contaminante |
| Vap | Vapor |

## Tabela 3 — Símbolos, Componentes de Código e Códigos Utilizados para Seleção de Respirador

| Símbolo | Descrição |
|---------|-----------|
| ¥ | Em concentrações acima do NIOSH REL, ou onde não houver REL, em qualquer concentração detectável |
| § | Entrada emergencial ou planejada em concentrações desconhecidas ou em condições IDLH |
| * | Substância relatada como causadora de irritação ou dano ocular; pode exigir proteção ocular |
| £ | Substância causa irritação ou dano ocular; é necessária proteção ocular |
| ¿ | Apenas sorventes não oxidáveis permitidos (não carvão ativado) |
| † | Indicador de fim de vida útil (ESLI) requerido |
| APF | Fator de proteção atribuído (FPA) |

| Código | Componente — Descrição |
|--------|------------------------|
| 95 | Respirador ou filtro para particulados com 95% de eficiência. Ver Tabela 4 (página xxv) para selecionar N95, R95 ou P95. |
| 99 | Respirador ou filtro para particulados com 99% de eficiência. Ver Tabela 4 (página xxv) para selecionar N99, R99 ou P99. |
| 100 | Respirador ou filtro para particulados com 99.97% de eficiência. Ver Tabela 4 (página xxv) para selecionar N100, R100 ou P100. |
| Ccr | Respirador com cartucho químico (chemical cartridge respirator) |
| F | Peça facial inteira (full facepiece) |

| Sa | Respirador de adução de ar (supplied-air respirator) |
| Scba | Equipamento de respiração autônomo (self-contained breathing apparatus) |
| Ag | Cartucho ou canister para gás ácido (acid gas) |
| Cf | Modo de fluxo contínuo (continuous flow) |

| Ov | Cartucho ou canister para vapor orgânico (organic vapor) |
